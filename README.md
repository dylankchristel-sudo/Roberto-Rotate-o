# Roberto-Rotate-o
Cool-vibes rotating robot designed to spread whimsy and joy by spinning!!!

# Features
2 opposite-facing motors turning wheels, creating spin
2 supporting heels for balance
Arduino Uno for brains
Metal frame
on/off switch(in progress)
