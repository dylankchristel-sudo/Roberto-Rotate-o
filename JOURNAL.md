# Day 1
To be honest, I knocked this out in one afternoon to meet a deadline, so it's all one day.

I started by disassembling some old robots I had around. I took the motors and motor controllers off and tested them to see if they still worked.
Then I reattached the wheel to each motor to see if it could still bear a load.
At this point, I lost more parts than I expected.
Next, I wired them to my Arduino Uno and tried to turn them on. This took a while because I kept writing the code completely wrong and had to rewire everything several times.
Eventually, I got the right code and wiring!
I tried to add an on/off button and make Roberto completely battery powered, but I was missing the parts, knowledge, and time to get those features working in time.
Then I put together a custom metal case from the robot parts. And added two extra wheels for support. I taped everything down for cable management.
However, I forgot to make a case for the Arduino, battery, and breadboard.
This makes it very difficult to run without sending the Arduino flying.
But the important thing is that it works!
This was a long night, but I learned a lot and enjoyed the process!
